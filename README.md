## Bacchus DB

This database is inspired by the need for an embedded JSON based database in a p2p application I plan to build with a 
colleague. I am building it to learn how databases are architected. I try handle data integrity in a unique way: mutexes.

## Program Procedure
- logic for data types
- logic for writing data to database
- logic for creating files
- logic for writing to created files
- logic for reading one record in the db
- logic for reading all records in the db
- logic to delete one record from db
- logic to delete all record from db

## Methods Available In BacchusDB (Struct Methods)
- read
- read all
- delete
- delete all
- write
